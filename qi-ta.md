```
如 ‘change some style in bdc’ ： 修改了bdc的样式

‘fix scroll bug in mobileIndex’:  修复了mobileIndex滚动的bug

如使用中文描述，也请说明修改了什么问题
```

**2.提交代码前，请检查增加/修改的代码是否符合本规范**

\(1\)删除代码中调试用的console.log

\(2\)注意缩进，js和css均为两个空格

\(3\)实际情况中无用的并且已经注释掉的代码
